----------- Команди щодо роботи з записами в Адресній книзі ----------------------------------------------

add Name [Phone] [Birthday]          - створити запис Name. Параметри Phone та Birthday - не обовʼязкові.
change_name Name_1 Name_2            - змінити імʼя Name_1 на імʼя Name_2.
delete Name                          - видалити запис Name з адресної книги.
print Name                           - показати вміст запису Name з адресної книги.

set_phone Name Phone                 - додати номер Phone до запису Name.
change_phone Name Phone_1 Phone_2    - змінити телефон Phone_1 запису Name на Phone_2.
find_phone Phone                     - знайти користувача, якому належить номер Phone.
delete_phone Name Phone              - видалити номер Phone з запису Name.

set_birthday Name Birthday           - встановити день народження Birthday у користувача Name.
change_birthday Name Birthday        - змінити день народження користувача Name на Birthday.
find_birthday Birthday               - знайти користувача з днем народження Birthday.

----------- Загальні команди щодо роботи з Адресною книгою -----------------------------------------------

all                                  - показати зміст всієї адресної книги.
birthdays                            - показати дні народження, які відбудуться протягом наступних 7-мі днів.

----------- Загальні команди щодо роботи з ботом ---------------------------------------------------------

quit (close, exit)                   - закінчити роботу Assistance Bot.
help                                 - список команд.

----------- Команди щодо роботи з записами в режимі роботи з обраним записом (RecordMode) -----------------

record_mode Name                     - обрати запис Name для роботи в режимі RecordMode.
change_name Name                     - змінити імʼя запису на імʼя Name.
set_phone Phone                      - додати номер Phone до запису.
change_phone Phone_1 Phone_2         - змінити телефон Phone_1 на Phone_2.
delete_phone Phone                   - видалити номер Phone з запису.
set_birthday Birthday                - встановити день народження Birthday.
print                                - показати вміст запису з адресної книги.
help                                 - список команд в режимі RecordMode.
quit                                 - вийти з режиму RecordMode.
